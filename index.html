About natasha119.com/atlasvinc.cc

The machine under the hood

You're currently connected to a Dell PowerEdge R720 which sits in my room, maybe under my bed, maybe beside my dresser.
There's not much to say about the machine itself... specs-wise it's fairly standard.

  • CPUs: E5-2670 v2 (10C, 20T) x2
  • RAM: 132GiB DDR3
  • STORAGE: ~11TiB total. 8x 1.2TiB 5600RPM HDDs in RAID5, 1x 3TiB SeaGate HDD
  • GPU: Motherboard-integrated Matrox G200eR2 (heh)
  • OS: Proxmox Virtual Environment

What is ATLASVINC and who is Natasha119?

What can be found here?

Great question. Not much. But I suppose if you have the credentials, you could access my Nextcloud instance.
There's also a barely-used but soon-to-not-be-barely-used Jellyfin instance running here too.
In the backend of things there's a Windows Server VM which hosts an Active Directory server and hosts the SMB shares for the 8TB disk array, and a Lab VM which is dispensible for new projects/concepts. Whenever those come along, heh.
